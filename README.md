# SafeLanka-Mobile-App

Disaster Safety & Emergency Assistance Mobile Application

## Description
SafeLanka is a mobile application designed to help users during emergency situations such as floods, landslides, and other disasters.

## Features
- User Authentication
- SOS Emergency Alert
- Live Location Sharing
- Safe Location Map
- Offline Emergency Guide
- Missing Person Reporting
- AI Voice Assistant

## Technologies
- Flutter
- Dart
- Firebase
- Google Maps API
- Gemini API
